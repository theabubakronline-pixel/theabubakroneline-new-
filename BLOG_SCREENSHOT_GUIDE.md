## üé® **Screenshot Design Tips**

### **Visual Guidelines:**
1. **Use Clear Annotations:**
   - Add numbered circles (1, 2, 3, 4) for steps
   - Use arrows to point to important elements
   - Add subtle highlights/boxes around key features

2. **Consistency:**
   - Use the same browser/device theme throughout
   - Maintain consistent sizing and quality
   - Keep the same zoom level for all screenshots

3. **Polish:**
   - Remove sensitive information (if any)
   - Use a clean, uncluttered browser view
   - Consider adding subtle shadows/borders

4. **Alt Text:**
   - All screenshots already have descriptive alt text
   - Make sure images match the alt text descriptions

### **Tools for Creating Screenshots:**
- **Mac:** Cmd + Shift + 4 (selection), Cmd + Shift + 3 (full screen)
- **Windows:** Snipping Tool, Snip & Sketch, or Windows Key + Shift + S
- **Chrome Extension:** Nimbus Screenshot, Awesome Screenshot
- **Online:** CleanShot, Monosnap

### **Editing Tools:**
- **Free:** GIMP, Paint.NET, Photopea (online)
- **Paid:** Photoshop, Sketch, Figma
- **Simple:** Canva (for adding annotations)
